# nhentaiDownloader
Projetinho para baixar direto do nhentai

# Como funciona

Abra o executavel, ele vai pedir o código do mangá que deseja baixar, se utiliza o nhentai vai saber que código é esse ( ͡° ͜ʖ ͡°).
